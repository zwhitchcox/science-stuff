
Notation matrix for reference:
![Notation Matrix](https://i.imgur.com/bDTAuKY.png)

```
repetition allowed:
    order matters:
        `n**k`
    order doesn't matter:
        `n! / (n - k)!`
repetition not allowed:
    order matters:
        n! / k!(n - k)!
    order doesn't matter:
        (n + k - 1)! / k!(n - 1)!
```
